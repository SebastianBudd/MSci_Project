This folder contains all the files required to run TBE. 

- ctrl.128 is the control file for the cell.

- pos0.128 is the initial positions of all the atoms.

- CONST22062016, ME22062016_OO and SPEC22062016_OO are include files for the control file.

- 0submitmag, 0submitti and 0submitopal are scripts to queue TBE on the three computors.

- 1submit is the script to queue find_h3o.py on tar.

- 0run_tbe is the script to run TBE
